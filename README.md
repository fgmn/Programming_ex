# Programming_ex
SDU 程序设计思维与实践实验代码

## Code
Code文件夹下是代码，没有题目，只能是在做某一道题时为大家提供思路，通过文件夹名称前缀索引(根据最长前缀原则匹配哦)：
|文件前缀|对应内容|
|-|-|
|H|作业题，和本周学习内容相关，一共15次|
|S|每月模拟题，一共4道|
|SS|模拟题测试，一共3次|
|T|模测(一共四道题，对标CSP的T1，T2，T4，但是没有那么难)，一共4次|

a,b,c,d四个文件夹是月模拟和模测的实验报告，H1-15是作业的，实验报告记录了一些题的思路，值得一看。
